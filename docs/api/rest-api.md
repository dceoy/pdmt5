# REST API (Under Development)

The REST API for **pdmt5** is planned but not implemented yet. This page
summarizes the current status and points to the specification and planning
materials that describe the intended behavior.

## Status

- The API is in the specification/design phase.
- There is no runnable FastAPI application in the package yet.
- Endpoints, configuration, and usage examples will be published once
  implementation starts.
